# Conexión del sandbox A2A con Gemini CLI

El sandbox expone:
  - Agent card : http://localhost:8080/.well-known/agent.json
  - gRPC       : lf.a2a.v1.A2AService en el puerto 8080

## Opciones de configuración

### Opción 1 — agent_card_url (requiere servidor arriba)

Coloca `sandbox.md` en:
  - Solo este proyecto : .gemini/agents/sandbox.md  (raíz del repo)
  - Todos los proyectos: ~/.gemini/agents/sandbox.md

Usa el fichero `sandbox-url.md` de este directorio como plantilla.

### Opción 2 — agent_card_json inline (sin servidor)

Usa el fichero `sandbox-inline.md` de este directorio como plantilla.
El JSON es el volcado exacto de AgentCardFactory.create().

## Comandos útiles en Gemini CLI

  /agents list             — ver agentes registrados
  /agents reload           — recargar el registro
  /agents enable a2a-sandbox
  /agents disable a2a-sandbox
